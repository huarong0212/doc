# 任务接口

### 一、用户任务明细接口

http:// 域名 / v1/task/task_list

| 参数    | 参数名 | 值    |
| ------- | ------ | ----- |
| user_id | 用户ID | 30459 |

返回值：json

{
"code":1,
"msg":"获取任务列表成功",
"data":{
	“list”:[
    		{
            "is_done":0,
            "task_id":2,
            "name":"上传任务",
            "desp":"上传新书得Q币",
            "src":"newusertask_1.jpg"
            },
			]	
	}
}

is_done 任务是否完成
task_id  任务id
name 任务名称
desp  任务描述
src  任务图标
